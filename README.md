# CHS Boys Soccer Program Management Platform

Official digital management platform for the Covington High School Boys Soccer Program.

- **School:** Covington High School — Covington, Louisiana
- **Program:** Boys Soccer — District 6-I
- **Head Coach & Program Director:** Warren Guillot, Jr.
- **Platform:** Built with HTML, CSS, and JavaScript
- **Database:** Google Sheets
- **Hosting:** GitHub Pages

## Platform Files

| File | Purpose |
|---|---|
| `index.html` | Login page — entry point for all users |
| `player-select.html` | Player/parent name selection after login |
| `coach.html` | Full coach management dashboard |
| `player.html` | Player and parent portal |
| `style.css` | Master stylesheet — all colors, layout, components |
| `app.js` | Core application logic, data fetching, utilities |
| `assets/` | Logo images and other static files |

## Access

- **Platform URL:** https://warrenguillotjr.github.io/chsboyssoccer
- **Public Site:** https://sites.google.com/view/chsboyssoccerprogram

## Modules

This platform manages roster, scheduling, player availability,
statistics, communication, equipment, tournaments, and program history
for the CHS Boys Soccer Program.
