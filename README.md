# CHS Boys Soccer Program Management Platform

Official digital management platform for the Covington High School Boys Soccer Program.

- **School:** Covington High School — Covington, Louisiana
- **Program:** Boys Soccer — District 6-I
- **Head Coach & Program Director:** Warren Guillot, Jr.
- **Platform:** Built with HTML, CSS, and JavaScript
- **Database:** Google Sheets
- **Hosting:** GitHub Pages

This platform manages roster, scheduling, player availability, 
statistics, communication, equipment, tournaments, and program history
for the CHS Boys Soccer Program.
